# 聊天助手 (Chat Assistant)

> 🌟 **人性化的 AI 陪伴伙伴** - 基于情绪感知的智能对话系统

## 📖 功能概述

聊天助手是一个注重**人性化体验**和**情感陪伴**的 AI 对话功能，不同于传统的效率导向型聊天工具，它专注于：

- 📊 **情绪追踪与分析** - 量化用户情绪变化，生成情绪日历
- 🎭 **个性化定制** - 丰富的个性化设置，打造专属 AI 伙伴
- 🌅 **智能问候** - 基于时间、节日和用户状态的主动问候
- 💬 **自然对话** - 支持情绪标注的对话体验

## 🎨 设计理念

遵循 **frontend/CLAUDE.md** 的设计规范：

1. **极简主义** - 清爽的界面，聚焦对话本身
2. **流体美学** - 使用 GSAP 实现平滑的过渡动画
3. **毛玻璃效果** - 在侧边栏和设置面板中使用 backdrop-filter
4. **微交互** - 按钮点击、悬浮反馈等细节动画
5. **未来主义** - 情绪可视化的数据呈现

## 🔧 核心功能

### 1. 情绪日历

- 📅 **可视化展示** - 使用热力图方式展示每日情绪状态
- 🎨 **颜色编码** - 根据情绪类型使用不同颜色
- 📊 **趋势分析** - 支持月份切换，查看情绪变化趋势
- 💡 **智能建议** - 根据情绪状态提供温馨建议

### 2. 主动问候系统

基于多种触发条件的智能问候：

- ⏰ **时间问候** - 早上/下午/晚上/深夜的时段问候
- 🎉 **节日问候** - 春节等传统节日的特别问候
- 📆 **特殊日期** - 周末、工作日等不同情境的问候
- ❤️ **情绪关怀** - 根据用户情绪状态的贴心问候

### 3. 个性化配置

#### 基本设置
- 助手昵称和头像
- 用户昵称和头像

#### 性格特征
- 🌸 **温柔体贴** - 温暖细致，像春风一样
- 👔 **专业理性** - 逻辑清晰，专业高效
- 😄 **幽默风趣** - 轻松幽默，妙语连珠
- 📚 **博学多才** - 知识渊博，见解独到

#### 回复风格
- 简洁明了
- 详细丰富
- 创意表达

#### 功能偏好
- 智能问候开关
- 情绪追踪开关
- 消息通知开关

### 4. 对话体验

- 😊 **情绪标注** - 发送消息时可选择当前情绪
- 📈 **实时分析** - AI 自动分析消息中的情绪关键词
- 💭 **智能回复** - 根据用户情绪生成贴心的回复
- 🎨 **精美 UI** - 符合设计规范的现代界面

当前使用 Mock 数据，未来可替换为真实 API：

## 🎯 使用场景

1. **情感陪伴** - 孤单时找人聊天，获得情感支持
